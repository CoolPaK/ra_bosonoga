# Дипломный проект по курсу «React: Библиотека №1 в современной фронтенд-разработке»

Это работающее приложение, всеми основными функциями которого можно пользоваться.

Реализованные механики:
  * постраничная навигация;
  * вывод хитов продаж и каталога с категориями;
  * поиск товаров (поиск в шапке перебрасывает на каталог);
  * добавление в корзину (с сохранением в localeStorage);
  * оформление заказа.
  
Используемые технологии:
  * React - React Router;
  * Redux - Redux Saga.

Для запуска приложения воспользуйтесь командами:
  * npm install
  * npm run start

Для подкулючения бекенда, откройте отдельно папку Backend в IDE и выполните команды:
  * npm install
  * npm run start
